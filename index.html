    What's new in .NET 5

    .NET 5.0 is the next major release of .NET Core following 3.1. We named this new release.NET 5.0 instead of .NET Core 4.0 for two reasons: We skipped version numbers 4.x to avoid confusion with .NET Framework 4.x. We dropped "Core" from the name to emphasize that this is the main implementation of .NET going forward.
